# Python Studies - files-.py

Este repositório contém arquivos Python organizados por datas de aulas, cobrindo conceitos fundamentais de programação em Python aplicados a estudos de ciência de dados.

## Estrutura do Diretório

### aula-2026.03.19
- **media.py**: Script simples para calcular a média de duas notas, primeiro com valores fixos e depois com entrada do usuário.
- **exercicios-19.03.py**: Conjunto de funções para exercícios básicos, incluindo cálculo de média de três notas, conversão de temperatura Celsius para Fahrenheit, cálculo de IMC, área de triângulo e preço com desconto.

### aula-2026.04.09
- **estruturadecisao.py**: Exemplos de estruturas de decisão, incluindo uso de dicionários para mapear dias da semana e estruturas condicionais.
- **estruturascondicionais.py**: Algoritmos com estruturas condicionais para aprovação/reprovação baseada em média, verificação de números pares/ímpares, conversão para positivo, e identificação de vogais/consoantes.
- **operacoesmatematicas.py**: Programa interativo para operações matemáticas básicas (soma, subtração, multiplicação, divisão) com escolha do usuário.
- **vogal.py**: Script para verificar se uma letra é vogal ou consoante, utilizando lista de vogais.

### aula-2026.04.16
- **estruturarepeticaowhile.py**: Exemplos de loops while para geração de tabuadas, com controle de usuário para continuar ou parar.
- **exercicios.py**: Exercícios utilizando while loops, incluindo contagem de números, verificação de idades, soma até zero, e identificação do menor número.

### aula-2026.04.23
- **estruturarepeticaofor.py**: Introdução a loops for, iterando sobre strings e ranges para tabuadas.
- **exercicios.py**: Exercícios com for loops, incluindo exibição de ranges, números ímpares, números aleatórios, raízes quadradas e identificação de maior/menor valor.

### EDAs
- **oque_e_eda.txt**: Explicação teórica e exemplos de código para Análise Exploratória de Dados (EDA), utilizando bibliotecas como pandas, numpy, matplotlib e seaborn.
- **sweetviz.py**: Script para gerar relatórios automatizados de EDA usando a biblioteca Sweetviz.

## Tópicos Estudados
- Entrada e saída de dados (input/output)
- Estruturas de decisão (if-else, match-case)
- Operadores relacionais e lógicos
- Loops de repetição (while e for)
- Funções básicas
- Operações matemáticas
- Análise Exploratória de Dados (EDA)
- Uso de bibliotecas como pandas, sweetviz, matplotlib, seaborn

## Como Usar
Cada arquivo pode ser executado individualmente em um ambiente Python. Para os scripts que requerem entrada do usuário, execute-os no terminal ou IDE.

Para EDA, certifique-se de ter as bibliotecas instaladas:
```
pip install pandas numpy matplotlib seaborn sweetviz
```
