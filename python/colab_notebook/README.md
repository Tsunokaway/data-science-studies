# 📓 Python-DS Projeto Inicial

Bem-vindo ao **Python-DS Projeto Inicial**! Este notebook Jupyter é o ponto de partida para seus estudos em Ciência de Dados com Python. Aqui você encontrará exercícios práticos e conceitos fundamentais para construir uma base sólida em programação Python aplicada à análise de dados.

## 🎯 Sobre o Notebook

Este notebook foi desenvolvido como um projeto inicial para estudantes de Ciência de Dados que estão começando sua jornada com Python. Ele combina teoria, prática e exercícios interativos para facilitar o aprendizado.

## 📚 Conteúdo Abordado

### 🛠️ Instalação e Configuração
- Instalação de bibliotecas com `pip`
- Gerenciamento de versões de pacotes
- Configuração do ambiente de desenvolvimento

### 📦 Bibliotecas Essenciais
- **Matplotlib**: Criação de gráficos e visualizações
- **NumPy**: Computação numérica e arrays
- **Random**: Geração de números aleatórios e amostragem

### 🔧 Conceitos de Programação
- **Funções**: Criação e uso de funções personalizadas
- **Escopo de variáveis**: Entendendo o alcance das variáveis
- **Retorno de valores**: Como as funções retornam resultados

### 📊 Exercícios Práticos
- Cálculos matemáticos e estatísticos
- Manipulação de listas e dicionários
- Visualização de dados básicos
- Desenvolvimento de algoritmos simples

## 🚀 Como Usar

### Opção 1: Google Colab (Recomendado)
1. Clique no botão abaixo para abrir no Colab:
   [![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/Tsunokaway/Projetos-pessoais/blob/main/Python-DS_Projeto-Inicial)

4. Experimente modificar os códigos

### Opção 2: Ambiente Local
1. Certifique-se de ter Python 3.x instalado
2. Instale as dependências:
   ```bash
   pip install matplotlib numpy jupyter
   ```
3. Abra o notebook:
   ```bash
   jupyter notebook Python-DS_Projeto-Inicial.ipynb
   ```
